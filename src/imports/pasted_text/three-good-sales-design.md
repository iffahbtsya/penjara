=================================================
CART SYNCHRONIZATION
====================

If a product is added to cart:

The SAME product must automatically appear in:

* Cart
* Checkout
* Payment
* Invoice
* Order Tracking
* Review Page

Displayed Data Must Always Match:

* Product Image
* Product Name
* Quantity
* Unit Price
* Subtotal

=================================================
QUANTITY SYNCHRONIZATION
========================

If quantity changes in Cart:

Automatically update:

* Checkout
* Payment
* Invoice

Example:

Cart:
Chair x2

Checkout:
Chair x2

Payment:
Chair x2

Invoice:
Chair x2

Must remain identical.

=================================================
REMOVE PRODUCT SYNCHRONIZATION
==============================

If user removes product from Cart:

Automatically remove from:

* Checkout
* Payment
* Invoice

=================================================
CUSTOM CLOTHING SYNCHRONIZATION
===============================

Custom Clothing Orders Must Behave Exactly Like Standard Products.

Custom Clothing:
→ Cart
→ Checkout
→ Payment
→ Invoice
→ Order Tracking
→ Review

Must remain connected throughout.

REWARD POINTS SYSTEM
====================

Points Granted:
Only After Order Status = Delivered

Checkout:

Save Reward Points

System Must:

* Recalculate Total
* Update Remaining Points

Customers Can:

* Earn Points
* Save Points
* Redeem Points

Customers Cannot:

* Use More Points Than Available
* Use Points Greater Than Order Value
